# D

Clist: 491

## 题意

黑板上一开始有一个整数 $X$.

接下来有 $Q$ 次操作. 第 $i$ 次操作给出两个整数 $A_i, B_i$，把这两个整数写到黑板上.

此时黑板上一共有 $2i+1$ 个整数，也就是初始的 $X$ 加上前 $i$ 次操作写入的 $2i$ 个数.

每次操作后，输出当前黑板上所有整数的中位数.

$1 \le Q \le 2 \times 10^5$

## Sol

注意到每次加两个整数，中位数位置最多移动一位。对左半维护一个大根堆，右半维护一个小根堆，每次加数对堆首比较判断即可。

复杂度 $O(Q \log Q)$.

# E

Clist: 1270

题意简述：

给定三个整数 $X_1, X_2, X_3$.

要求统计有多少个长度为 $X_1+X_2+X_3$ 的序列 $A$，满足：

1. 序列中恰好有 $X_1$ 个 $1$，$X_2$ 个 $2$，$X_3$ 个 $3$；
2. 相邻两个数的差的绝对值不超过 $1$，也就是不能出现相邻的 $1$ 和 $3$.

输出满足条件的序列数量，对 $998244353$ 取模.

# Sol

考虑满足条件的序列模样，共两种情况：
1. $\{12\}\{2\}\{23\}$的重复
2. $\{23\}\{2\}\{12\}$的重复

$\{i\}$ 指的是含 $i$ 的任意非 0 长度串.

观察发现，序列可看为在 $1$, $3$ 中插入 $2$.

我们考虑第一种情况, 因为第二种情况与第一种情况类似.

枚举 $\{2\}$ 的数量, 对于 $i$ 个 $\{2\}$, 有 $\lceil \frac{i+1}{2} \rceil$ 个 $\{12\}$ 和  $\lfloor \frac{i+1}{2} \rfloor$ 个 $\{23\}$. 

1. 先考虑 $1$ $3$ 排列方案，即“将 $X_1$ 个球放入 $\lceil \frac{i+1}{2} \rceil$ 个箱子和 $X_3$ 个球放入 $\lfloor \frac{i+1}{2} \rfloor$ 个 $\{23\}$ 个箱子，箱子都非空”的方案数. 即 $\binom{X_1 - 1}{\lceil \frac{i+1}{2} \rceil - 1} \binom{X_3 - 1}{\lfloor \frac{i+1}{2} \rfloor - 1}$
2. 在考虑 $2$ 排列方案，即“将 $X_2$ 个球放入 $X_1+X_3+1$ 个箱子里，其中 $i$ 个箱子非空其他箱子可空”的方案数. 即 $\binom{X_2 + X_1 + X_3-i}{X_1+X_3}$

第一种情况的方案总数即为 $\sum \binom{X_1 - 1}{\lceil \frac{i+1}{2} \rceil - 1} \binom{X_3 - 1}{\lfloor \frac{i+1}{2} \rfloor - 1}\binom{X_2 + X_1 + X_3-i}{X_1+X_3}$. 第二种情况同理.

复杂度 $O(X_1+X_2+X_3)$.
